# Laundry Expert - Landing Page

A responsive, lightweight landing page for a premium laundry and dry-cleaning service. Built completely with semantic HTML5 and clean, modern CSS3 (Flexbox), featuring a native, JavaScript-free mobile responsive navigation menu.

## Features

- **Modern Hero Section:** Split-screen layout highlighting primary services with clean visual hierarchies and custom call-to-action buttons.
- **Pure CSS Mobile Navigation:** Uses the CSS checkbox-hack technique to handle sidebar menu toggling on mobile devices without importing bulky JavaScript engines.
- **Fully Responsive:** Adapts seamlessly across desktop monitors, tablets, and smartphones using CSS Media Queries.
- **Interactive Micro-animations:** Includes dynamic transitions on hover actions for interactive UI elements.
