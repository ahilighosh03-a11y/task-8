# Thala 7 - Static Sneaker Storefront Interface

A responsive, single-page storefront web layout created strictly with standard HTML5 and CSS3 features.

## 📁 Project Structure
* `index.html` - The webpage structures containing the mock views.
* `style.css` - UI custom variables, grid layouts, and the native anchor CSS router.

## 💡 Implementation Decisions (No JavaScript)
Per assignment milestones targeting strictly presentation layer structures, this project deliberately omits scripting languages:
1. **CSS Anchor Routing (`:target`):** The application handles view transitions entirely via native URL fragment hashes matching standard element IDs.
2. **Static Mock Elements:** In place of functional calculations, interactive items like quantity controls are presented as clean, static textual indicators to accurately represent layout placement without requiring backend calculations.
